Typhoon

What is Typhoon?

Typhoon is a webserver. It's built to be fast (and also parses basic PHP as a little extra). It's written in Ruby and it uses YAML as config file format.

How to use?

It's simple, just make your config in config.yml (the keys are self explainable) and fire up Terminal and run ./boot.rb start on the cloned directory. You can use ./boot.rb start -t instead if you want to log everything (basically it doesn't daemonizes).

What gems are needed?

Not much, just one (for now):

  • daemons: gem install daemons
